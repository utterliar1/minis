📦 安装方法

1
解压文件夹
将 CadToolkit 文件夹解压到 C:\CadToolkit（或任意位置）
2
加载插件
在 CAD 命令行输入：
(load "C:/CadToolkit/autoload.lsp")
看到 CadToolkit v1.24 ready. Type CC to start. 提示即成功
3
开始使用
输入 CC 呼出工具面板，点击按钮执行命令
💡 推荐：将 (load "C:/CadToolkit/autoload.lsp") 添加到 CAD 启动组，每次打开 CAD 自动加载，直接输入 CC 即可。

📝 基本操作

打开面板

命令行输入 CC，弹出分组工具面板。

关闭面板

  • 按 ESC 键
  • 点击右上角 X

预选模式

先在图线中选中对象，再输入 CC 打开面板点击命令，选中的对象会自动作为命令的操作对象。

两种执行方式

  • 面板点击：输入 CC → 点击按钮
  • 命令行：直接输入命令名（如 CT_ALIGN）

✍️ 文字编辑

6 个命令

查找替换 CT_FINDREPLACE

批量替换图线中的文字内容。

  • 支持单行文字、多行文字、块属性文字
  • 支持块定义内部的文字替换
  • 可选忽略大小写

文字对齐 CT_ALIGN

将多个单行文字整齐排列。

  • 水平方向：左对齐 / 居中 / 右对齐
  • 基点方式：第一个选中文字 / 手动指定
  • 行距：自动（根据字高计算）/ 手动输入
  • 对齐设置自动保存，下次使用自动恢复

加下划线 CT_UNDERLINE

将单行文字转换为带下划线的多行文字，保持原始内容和位置不变。

格式复制 CT_TEXTBRUSH

文字格式刷。选择源文字获取格式（图层、颜色、字高、字体），再选择目标文字一键应用。

文字合并 CT_TEXTMERGE

将多个文字按从上到下的顺序合并为一个多行文字。字高自动继承源文字。

文字编号 CT_TEXTNUMBER

为选中的文字添加顺序编号。

  • 追加模式：在原文字后加编号（如「房间」→「房间1」「房间2」）
  • 替换模式：用编号替换原文字（如「任意文字」→「1」「2」「3」）
  • 可自定义前缀、后缀、起始编号

👻 图层管理

5 个命令

图层归零 CT_SETLAYER0

将选中对象全部移到图层 0，快速清理杂乱图层。

  • 递归处理块定义内部对象
  • 颜色同时改为随层

图层规范 CT_LAYERSTANDARD

按 [LayerStandard] 和 [LayerMap] 预览并规范当前空间图层。

  • 自动创建/修正标准图层的颜色、线型、线宽和打印状态
  • 将匹配到的旧图层对象移动到标准图层
  • 未匹配图层不会自动修改

孤立图层 CT_ISOLAYER

选中一个对象，冻结除该对象所在图层以外的所有图层。再次执行可恢复。

按层选择 CT_SELECTBYLAYER

先选源对象获取图层，再选择搜索范围；直接回车则搜索全图。

按色选择 CT_SELECTBYCOLOR

先选源对象获取颜色，再选择搜索范围；直接回车则搜索全图。

🧱 图块操作

4 个命令

重命名块 CT_RENAMEBLOCK

点选一个块参照，自动识别块名，在弹窗中输入新名称完成重命名。

快捷建块 CT_QUICKBLOCK

选择对象 → 指定基点 → 自动创建块。

  • 块名自动生成：前缀 + 递增编号（如 BK001、BK002）
  • 可配置是否删除原对象

改块基点 CT_CHANGEBASEPOINT

选择普通块参照，指定新的块基点，插件更新同名块定义并补偿现有参照位置，使图面上的块图形保持不动。

按块选择 CT_SELECTBYBLOCK

先选源块参照获取块名，再选择搜索范围；直接回车则搜索全图。

🖊️ 绘图标注

4 个命令

画中心线 CT_CENTERLINE

为选中的圆和矩形自动绘制中心线。

  • 图层：0，颜色：红色，线型：CENTER（自动加载）
  • 长度按图形尺寸自动延伸

快速标注 CT_QUICKDIM

选中对象后自动计算包围盒，一键创建水平 + 垂直两个方向的尺寸标注。

递增复制 CT_INCCOPY

选中含编号的对象，连续指定复制位置，每次复制自动递增编号。

  • 识别尾部数字自动递增（A001 → A002 → A003）
  • 回车结束

Z轴归零 CT_FLATTEN

将选中对象的 Z 坐标全部归零，将 3D 对象压平到 XY 平面。

  • 支持：直线、圆、圆弧、多段线、三维多段线、文字、块参照、标注、样条曲线
  • 其他对象通过包围盒位移归零

⚙️ 配置文件

CadToolkit.ini 统一放在 CadToolkit 根目录，AutoCAD / ZWCAD / GstarCAD 三端共享同一份配置。升级或重新部署时保留已有 CadToolkit.ini，发布包中的 CadToolkit.default.ini 仅作为默认模板。

如果已有 CadToolkit.ini 缺少新版新增的基础配置项，插件启动时会自动补上默认值；已有配置值、自定义 [Commands]、[LayerStandard] 和 [LayerMap] 不会被覆盖或合并。

配置项说明默认值
QuickBlockPrefix快捷建块前缀BK
DeleteOriginal建块后删除原对象true
KeepOriginal加下划线时保留原文字false
AlignHorizontal对齐方向（0左/1中/2右）0
AlignUseFirstBase以第一个文字为基点true
AlignLineSpacing对齐行距（0=自动）0
IsoLayerKeepLayer0孤立图层时是否保留 0 层false

自定义命令

在 [Commands] 段添加任意 CAD 命令：

# 格式：显示名称=CAD命令
清理=PU
打印=PLOT
图层管理=LAYMCOL

也可以通过面板底部 + / - 按钮添加和管理。

图层规范

图层规范由 CadToolkit.ini 控制。三端共用 C:\CadToolkit\CadToolkit.ini，修改后下一次执行 CT_LAYERSTANDARD 即可读取新配置。

  • 先按 [LayerStandard] 创建或修正标准图层属性
  • 再按 [LayerMap] 识别旧图层并迁移对象
  • 白名单图层保持原样，不参与规范化，也不会归到 0 层
  • 执行前会先预览，可用顶部筛选按钮和关键词框按图层名过滤查看，也可以点“复制当前”复制当前视图，确认后才会修改图纸

基础开关

LayerStandardFallbackTo0=false
LayerStandardWhitelist=0,Defpoints,*图框*,*视口*,*原有*,*新增*
配置项说明
LayerStandardFallbackTo0未识别图层是否归到 0 层。建议默认关闭，需要集中清理时在预览窗口临时勾选。
LayerStandardWhitelist全局白名单，支持精确匹配和 * 通配，多个规则用英文逗号分隔。

白名单写法

写法含义
0只匹配名为 0 的图层
Defpoints只匹配 Defpoints
*图框*图层名中包含“图框”
*原有*图层名中包含“原有”
A-*以 A- 开头
*-OLD以 -OLD 结尾

标准图层

[LayerStandard] 格式为：标准图层=颜色|线型|线宽|是否打印。

字段示例说明
颜色4、200CAD ACI 颜色号
线型CONTINUOUS、CENTER线型名，缺失时会尝试从 CAD 自带 lin 文件加载
线宽Default、0.25Default 表示默认线宽；数字按毫米写
是否打印true / false是否参与打印

旧图层识别

[LayerMap] 格式为：标准图层=别名1,别名2,别名3。别名默认全字匹配；需要包含、前缀、后缀匹配时用 * 通配符。

例如 0-4 只匹配名为 0-4 的图层，*风管* 才会匹配 风管-一层 并归到 5-风网。

完整示例

LayerStandardFallbackTo0=false
LayerStandardWhitelist=0,Defpoints,*图框*,*视口*,*原有*,*新增*

[LayerStandard]
0-设备层=4|CONTINUOUS|Default|true
1-中心线层=1|CENTER|Default|true
2-虚线层=4|HIDDEN|Default|true
3-文字层=3|CONTINUOUS|Default|true

[LayerMap]
0-设备层=*设备*,0-4,*VIS*
1-中心线层=*中心*,*中心线*,CENTER,0-1,1,AXIS,CLEARANCE
2-虚线层=*虚线*,HIDDEN,DASH
3-文字层=*文字*,*说明*,*编号*,TEXT,txt
5-风网=*风网*,*风管*,*风道*,0-5
建议第一次使用时保持 LayerStandardFallbackTo0=false。先看预览，把暂时不确定的图层加进白名单，确认无误后再执行。

❓ 常见问题

Q：输入 CC 没有反应？
A：确认已加载插件。输入 (load "C:/CadToolkit/autoload.lsp")，看到成功提示后再试。
Q：面板按钮点击后没有反应？
A：部分命令需要先选中对象再点击。支持预选模式：先选对象→再输入 CC →点击命令。
Q：配置文件修改后需要重新加载吗？
A：不需要重新加载 LSP，但需要重新输入 CC 打开面板才能看到更新。
Q：三个 CAD 可以同时使用吗？
A：可以。三个 CAD 共享同一份配置文件，修改一次三端同步。
Q：如何迁移到其他电脑？
A：将整个 CadToolkit 文件夹复制到新电脑，按安装步骤重新加载即可，所有配置都保留。
