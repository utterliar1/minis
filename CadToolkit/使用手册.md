# CadToolkit 使用手册

> 版本：v1.2 | 适用：AutoCAD / 中望CAD / 浩辰CAD

---

## 一、安装方法

### 1. 复制文件夹

将 `CadToolkit` 文件夹整体复制到电脑任意位置，推荐 `C:\CadToolkit`。

### 2. 首次加载

打开 CAD，在命令行输入：

```
(load "C:/CadToolkit/autoload.lsp")
```

看到 `CadToolkit loaded` 提示即加载成功。

### 3. 设置自动加载（推荐）

将上面的 `(load ...)` 命令添加到 CAD 的启动脚本中：

- **AutoCAD**：编辑 `acad.lsp` 或通过 `APPLOAD` → 启动组添加
- **中望CAD**：同上
- **浩辰CAD**：同上

设置后每次打开 CAD 直接输入 `CC` 即可使用。

## 二、基本使用

### 打开面板

在命令行输入 `CC`（两个字母），弹出工具面板。

面板按功能分组显示所有命令，点击按钮即可执行对应功能。

### 关闭面板

- 按 `ESC` 键
- 点击右上角 `X`

### 执行命令的两种方式

1. **面板点击**：输入 `CC` → 点击按钮
2. **命令行直接输入**：如果记住了命令名，可以直接在命令行输入（如 `CT_ALIGN`）

## 三、功能详解

### 文字编辑

#### 查找替换（CT_FINDREPLACE）

批量替换图纸中的文字内容。

1. 点击命令后弹出查找替换对话框
2. 输入查找内容和替换内容
3. 可勾选「忽略大小写」
4. 点击确定，自动替换所有匹配的文字

支持范围：单行文字、多行文字、块属性文字。

#### 文字对齐（CT_ALIGN）

将多个单行文字整齐排列。

1. 先选中需要对齐的多个单行文字
2. 点击命令，在弹窗中设置：
   - **水平方向**：左对齐 / 居中 / 右对齐
   - **基点方式**：以第一个选择的文字为基点 / 手动指定基点
   - **行距**：自动（根据字高计算）/ 手动输入具体数值
3. 点击确定完成对齐

对齐设置会自动保存，下次使用时自动恢复上次的配置。

#### 加下划线（CT_UNDERLINE）

为单行文字添加下划线效果。

1. 选中需要加下划线的单行文字
2. 点击命令，自动转换为带下划线的多行文字

转换后文字内容和位置不变，视觉上增加下划线。

#### 格式复制（CT_TEXTBRUSH）

文字格式刷，快速统一文字样式。

1. 点击命令
2. 选择源文字（获取其格式：图层、颜色、字高、字体样式）
3. 选择一个或多个目标文字
4. 目标文字的格式自动改为与源文字一致

#### 文字合并（CT_TEXTMERGE）

将多个文字合并为一个。

1. 选中多个单行文字或多行文字
2. 点击命令，所有文字按从上到下的顺序合并为一个多行文字

#### 文字编号（CT_TEXTNUMBER）

为文字添加顺序编号。

1. 选中多个文字
2. 选择模式：
   - **追加**：在原文字后面添加编号（如「房间」→「房间1」「房间2」）
   - **替换**：用编号替换原文字（如「任意文字」→「1」「2」「3」）
3. 输入起始编号和步长
4. 点击确定完成编号

---

### 图层管理

#### 图层归零（CT_SETLAYER0）

将选中对象全部移到图层 0。

1. 选中需要清理的对象
2. 点击命令，所有对象移到 0 层

适用场景：从外部图纸复制过来的对象图层杂乱，一键归零。

#### 孤立图层（CT_ISOLAYER）

临时只显示某个图层。

1. 选中一个对象
2. 点击命令，除该对象所在图层外，其他图层全部冻结
3. 再次执行命令，恢复之前冻结的图层

适用场景：需要在复杂图纸中专注某个图层的内容。

#### 按层选择（CT_SELECTBYLAYER）

快速选中同图层的所有对象。

1. 选中一个对象（或先选好再点命令）
2. 点击命令，自动选中当前图纸中同一图层的所有对象

#### 按色选择（CT_SELECTBYCOLOR）

快速选中同颜色的所有对象。

1. 选中一个对象
2. 点击命令，自动选中当前图纸中同一颜色的所有对象

#### 按块选择（CT_SELECTBYBLOCK）

快速选中同名块的所有实例。

1. 选中一个块参照
2. 点击命令，自动选中当前图纸中所有同名块参照

---

### 图块操作

#### 重命名块（CT_RENAMEBLOCK）

重命名已有的块定义。

1. 点击命令，在图纸中点选一个块参照
2. 弹窗自动显示当前块名（只读）
3. 输入新名称，点击确定完成重命名

无需手动输入旧块名，点选即可自动识别。

#### 快捷建块（CT_QUICKBLOCK）

快速将选中对象创建为块。

1. 选中需要建块的对象
2. 点击命令，指定块基点
3. 自动创建块，块名为配置的前缀 + 递增编号（如 BK001、BK002...）

可在配置文件中修改：
- `QuickBlockPrefix`：块名前缀（默认 BK）
- `DeleteOriginal`：建块后是否删除原对象（默认 true）

---

### 绘图标注

#### 画中心线（CT_CENTERLINE）

为圆和矩形自动绘制中心线。

1. 选中一个或多个圆/矩形
2. 点击命令，自动绘制中心线

中心线属性：
- 图层：0
- 颜色：红色
- 线型：CENTER（自动加载）
- 长度：按图形尺寸自动延伸

#### 快速标注（CT_QUICKDIM）

一键创建水平 + 垂直两个方向的尺寸标注。

1. 选中需要标注的对象
2. 点击命令，自动计算包围盒
3. 在下方创建水平标注，在右侧创建垂直标注

标注偏移量自动按图形比例计算。

#### 递增复制（CT_INCCOPY）

复制对象并自动递增编号。

1. 选中含编号文字的对象（如标有 A001 的对象）
2. 点击命令
3. 连续指定复制位置，每次复制后编号自动递增（A002、A003...）
4. 按回车结束

适用场景：房间编号、设备编号等需要批量递增的场景。

## 四、自定义命令

### 通过配置文件添加

编辑 `CadToolkit.ini`，在 `[Commands]` 段添加：

```ini
[Commands]
# 已有命令...
# 自定义命令
清理=PU
打印=PLOT
图层管理=LAYMCOL
```

格式：`显示名称=CAD命令名`

### 通过面板添加

1. 点击面板底部 `+` 按钮
2. 输入显示名称（如「清理」）
3. 输入 CAD 命令名（如 `PU`）
4. 确定后自动写入配置文件

### 管理已有命令

点击面板底部 `-` 按钮，可编辑或删除已有命令。

## 五、配置文件说明

配置文件 `CadToolkit.ini` 位于插件根目录，与 DLL 放在一起，方便整体迁移。

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `Version` | 版本号 | v1.2 |
| `QuickBlockPrefix` | 快捷建块的块名前缀 | BK |
| `DeleteOriginal` | 建块后删除原对象 | true |
| `KeepOriginal` | 加下划线时保留原文字 | false |
| `AlignHorz` | 文字对齐水平方向（0左/1中/2右） | 0 |
| `AlignUseFirstBase` | 对齐以第一个文字为基点 | true |
| `AlignLineSpacing` | 对齐行距（0=自动） | 0 |

## 六、常见问题

**Q：输入 CC 没有反应？**
A：确认已加载插件。输入 `(load "C:/CadToolkit/autoload.lsp")`，看到成功提示后再试。

**Q：面板按钮点击后没有反应？**
A：部分命令需要先选中对象再点击。建议先在图纸中选好对象，再输入 CC 点击命令。

**Q：配置文件修改后需要重新加载吗？**
A：不需要重新加载 LSP，但需要重新输入 CC 打开面板才能看到更新。

**Q：三个 CAD 可以同时使用吗？**
A：可以。三个 CAD 共享同一份配置文件 `CadToolkit.ini`，修改一次三端同步。

**Q：如何迁移到其他电脑？**
A：将整个 `CadToolkit` 文件夹复制到新电脑，按安装步骤重新加载即可，所有配置和自定义命令都保留。
