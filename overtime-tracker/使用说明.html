使用说明 v2.0 | 2026年6月

一、系统概述

考勤助手是一套基于 GPS 地理围栏 的轻量级考勤系统，支持多人团队使用。管理员可实时查看所有成员的打卡状态和工时统计，系统自动识别法定节假日和调休工作日，准确计算加班时长。

📍

GPS 地理围栏

必须在指定位置范围内才能打卡，防止代打卡，确保考勤真实性

📊

自动工时计算

自动识别工作日/休息日，精确计算加班时长，扣除午休时间

🎌

节假日同步

一键同步国家法定节假日和调休工作日，无需手动配置

📧

邮件报表

自动定时发送工时统计报表到指定邮箱，方便管理

📤

灵活导出

支持按人员、自定义日期范围导出 CSV 报表

🔒

权限分离

管理员与普通用户界面分离，数据隔离，安全可靠

三、使用流程

管理员初始化

1. 登录 → 2. 设置位置 → 3. 配置时间 → 4. 同步节假日 → 5. 添加成员
💡 提示：管理员账号默认密码建议首次登录后立即修改（设置 → 修改密码）。

成员注册与打卡

1. 管理员添加姓名到白名单 → 2. 成员用姓名注册账号 → 3. 到达打卡范围后打卡
📍 打卡说明：成员需在管理员设定的 GPS 范围内（默认 500 米）才能打卡。如因 GPS 漂移等原因在范围外，系统会提示确认，打卡记录将标记为"范围外"。

四、管理员界面详解

4.1 概览页

管理员首页显示 4 项核心指标（成员数、今日工时、本周工时、本月工时），以及所有成员的实时打卡状态表格。

近 30 天工时趋势

柱状图展示团队近 30 天的每日总工时变化趋势。

本月工时排行

按本月工时从高到低排列，显示团队平均值，超过平均 1.5 倍用红色标记。

4.2 导出报表

点击「📤 导出」弹出导出配置：

  • 人员：所有人 / 指定成员
  • 时间：今日 / 本周 / 本月 / 全部 / 自定义日期范围
  • 导出内容：姓名、日期、星期、上班时间、下班时间、类型、事由、工时

4.3 成员管理

白名单管理

添加姓名到白名单，删除未使用的白名单记录

已注册用户

切换角色、重置密码、删除用户（删除后白名单自动重置为可注册状态）

4.4 系统设置

设置项说明
打卡位置点击「📍 使用当前位置」自动获取 GPS 坐标
打卡范围100m / 200m / 300m / 500m / 1000m
工作时间上班 / 下班 / 午休时间
工作日选择哪些天是默认工作日
节假日同步一键同步法定节假日和调休工作日
修改密码修改当前账号密码

4.5 邮件配置

配置 SMTP

填写邮件服务器地址、端口、发件邮箱、授权码

添加收件人

每行一个邮箱地址，支持多个收件人

设置定时

开启定时发送，设置每日发送时间

测试发送

点击「📧 测试」确认配置正确

📧 邮件内容：系统会自动生成 HTML 格式的工时统计报表，包含所有成员的本月加班汇总表格。

五、成员界面详解

⏱

打卡页

显示北京时间、GPS 状态、打卡按钮。上班打卡需填写加班事由，下班打卡直接记录。

📅

记录页

日历视图查看打卡记录，点击任意日期弹出该日打卡详情（时间、事由、工时）。

📊

统计页

本周/本月/累计工时统计，月度趋势柱状图，支持导出个人报表。

工时计算规则

日期类型工时计算
工作日（超出下班时间）总在岗时间 - 标准工时(8h) - 午休 = 加班工时
休息日 / 法定节假日全部在岗时间均算加班工时
调休补班（周末补班）按工作日规则计算

七、常见问题

Q: 注册时提示"姓名不在管理员名单中"？

需要管理员先在「成员」页添加该姓名到白名单。

Q: 打卡时显示"GPS精度不足"？

手机 GPS 信号弱，建议到开阔地带等待精度恢复。

Q: 删除用户后白名单还是显示已注册？

v2.0 已修复：删除用户时白名单自动重置为可注册状态。

Q: 节假日数据过时了？

在设置页点击「📥 同步」重新获取最新数据。

Q: 如何备份数据？

数据存储在服务器 ./data/overtime.db，可直接拷贝该文件备份。
