# BlockBrowser 块浏览器 - 使用手册

## 快速开始

### 第一步：加载插件

1. 打开CAD（浩辰CAD / AutoCAD / 中望CAD）
2. 在命令行输入 `APPLOAD` 回车
3. 找到插件目录，选择 `autoload.lsp` 文件，点击「加载」
4. 命令行显示「块浏览器已就绪，输入 BB 启动。」即为成功

> 提示：将 `autoload.lsp` 添加到CAD启动脚本可实现自动加载，无需每次手动加载。

### 第二步：打开块浏览器

在命令行输入 `BB`（或 `KLLQ`）回车，弹出块浏览器窗口。

## 功能操作

### 浏览块

- **切换分类**：点击分类栏的按钮切换不同分类
- **搜索**：在搜索框输入关键字，实时过滤块名称
- **缩略图大小**：右侧下拉框可选「小/中/大/特大」
- **预览**：缩略图自动显示DWG文件内容，首次加载需要几秒，之后有缓存

### 插入块

1. 在缩略图网格中**单击**选中要插入的块（蓝色边框表示选中）
2. 点击工具栏「插入」按钮，或直接**双击**缩略图
3. 弹出对话框设置「比例」和「角度」，点击确定
4. 关闭对话框后，在CAD图纸中**点击**指定插入点
5. 块即插入到指定位置

> 也可以不选中直接双击缩略图快速插入。

### 添加到库（保存选中的图形为块）

1. 点击工具栏「添加到库」
2. 在弹窗中选择**分类**，输入**块名称**，点击确定
3. 在CAD图纸中**点击**指定块的基点（插入时的对齐点）
4. **框选**要保存的图形对象，回车确认
5. 块即保存到对应分类目录

### 导出块（从当前图纸导出已有块）

1. 点击工具栏「导出块」
2. 弹窗列出当前图纸中**所有块定义**
3. 选择要导出的块，选择目标分类，点击「导出」
4. 块即保存到对应分类目录

### 设置

点击「设置」按钮可修改块库路径。默认路径为插件目录下的 `我的常用块` 文件夹。

## 块库目录结构

块库是一个普通文件夹，按子文件夹分类：

```
我的常用块/
+-- 常用/          <- 常用的块
+-- 电气/          <- 电气相关
+-- 建筑/          <- 建筑相关
+-- 机械/          <- 机械相关
+-- 标注/          <- 标注符号
+-- 其他/          <- 未分类
    +-- 某个块.dwg  <- 每个DWG文件就是一个块
    +-- 另一个块.dwg
```

- 直接在文件夹里放入DWG文件即可被块浏览器识别
- 可以新建子文件夹作为新分类，块浏览器自动识别
- 删除DWG文件即从库中移除

## 常见问题

### Q: 加载autoload.lsp后输入BB没反应？

确认命令行显示了「块浏览器已就绪」。如果没有，尝试手动加载DLL：
- 浩辰CAD：输入 `NETLOAD`，选择 `gcadBlockBrowser.dll`
- AutoCAD：输入 `NETLOAD`，选择 `acadBlockBrowser.dll`
- 中望CAD：输入 `NETLOAD`，选择 `zwcadBlockBrowser.dll`

### Q: 缩略图不显示？

1. 首次加载需要时间，等状态栏显示「就绪」
2. 点击「刷新」按钮清除缓存重新生成
3. 确认DWG文件未损坏（能正常用CAD打开）

### Q: 插入块时提示「导入失败」？

1. 确认DWG文件是有效格式（非DXF改名）
2. 检查文件是否被其他程序占用
3. 尝试在CAD中先用 `INSERT` 命令手动插入测试

### Q: 如何共享块库给团队？

将 `我的常用块` 文件夹放在网络共享路径，然后在设置中修改块库路径为网络路径即可。也可以把整个插件目录放在共享盘。

## 命令列表

| 命令 | 说明 |
|------|------|
| BB | 打开块浏览器 |
| KLLQ | 打开块浏览器（快捷别名） |

| BBEXPORT | 命令行方式导出块 |
| BBTHUMB | 清除所有缩略图缓存 |
