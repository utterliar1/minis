# minis

个人脚本、CAD 插件和自动化工具合集。

## 项目索引

| 目录 | 内容 | 入口 |
|------|------|------|
| `CadToolkit/` | AutoCAD / 中望 CAD / 浩辰 CAD 三平台通用工具箱插件。一个 `CC` 命令呼出分组面板，包含文字、图层、图块、绘图标注等工具。 | [CadToolkit/README.md](CadToolkit/README.md) |
| `BlockBrowser/` | 多 CAD 平台可视化块浏览器插件，包含块库、缩略图和三平台 DLL。 | [BlockBrowser/README.md](BlockBrowser/README.md) |
| `overtime-tracker/` | 加班、打卡与工时统计相关工具。 | [overtime-tracker/README.md](overtime-tracker/README.md) |
| `nodeseek-checkin/` | NodeSeek 论坛自动签到脚本。 | [nodeseek-checkin/README.md](nodeseek-checkin/README.md) |
| `xmSport/` | 小米运动刷步数相关脚本。 | [xmSport/README.md](xmSport/README.md) |
| `jbot/` | Telegram 机器人插件与扩展脚本。 | [jbot/README.md](jbot/README.md) |
| `telegram-bot/` | Telegram 机器人相关脚本。 | [telegram-bot/README.md](telegram-bot/README.md) |

## 目录结构

```text
minis/
├── BlockBrowser/          CAD 块浏览器插件
├── CadToolkit/            CAD 工具箱插件
├── jbot/                  Telegram 机器人插件
├── nodeseek-checkin/      NodeSeek 签到脚本
├── overtime-tracker/      工时统计工具
├── telegram-bot/          Telegram 机器人脚本
└── xmSport/               小米运动脚本
```
